## Welcome to YagoutPay SDK

YagoutPay SDK provides comprehensive, multi-language SDKs for integrating with the YagoutPay payment gateway. Our SDKs support both **Hosted Form** and **Direct API** integration methods across multiple platforms.

## Key Features

<AccordionGroup>
  <Accordion title="Spec-Accurate Assembly">
    Our SDKs build `merchant_request` in the exact sectioned format required by the YagoutPay gateway specification, ensuring compatibility and reliability.
  </Accordion>
  <Accordion title="Modern Cryptography">
    SHA-256 hashing with AES-256-CBC encryption using PKCS7 padding and static IV for secure data transmission.
  </Accordion>
  <Accordion title="Multiple Integration Methods">
    Support for Hosted Form (redirect-based) and Direct API (server-to-server) integration patterns.
  </Accordion>
  <Accordion title="Payment Links">
    Generate static and dynamic payment links for flexible payment collection scenarios.
  </Accordion>
</AccordionGroup>

## Quick Start

Choose your platform and get started in minutes:

<Steps>
  <Step title="Install SDK">
    Add the YagoutPay SDK to your project using your platform's package manager
  </Step>
  <Step title="Configure Credentials">
    Set up your merchant ID and encryption key securely
  </Step>
  <Step title="Build Payment Request">
    Use the SDK to assemble your payment request with proper encryption
  </Step>
  <Step title="Process Payment">
    Submit to YagoutPay gateway and handle the response
  </Step>
</Steps>
